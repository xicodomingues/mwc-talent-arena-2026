About this site

An unofficial schedule viewer for MWC Talent Arena 2026 (March 3–4, Barcelona).

Features

  • Three views: List, Calendar grid, and Timeline
  • Filter by day, stage, topic, and language
  • Search sessions by title, speaker, or description
  • Hide sessions you're not interested in
  • Live "now" indicator during the event
  • Smart calendar export matching your current filters and view

How it was built

This project was vibecoded with Claude (Opus) using Claude Code. It's a single HTML file with no frameworks or build steps.

Source

The code is open source on GitHub.
